# yachtee

[![build](https://github.com/mosmeh/yachtee/workflows/build/badge.svg)](https://github.com/mosmeh/yachtee/actions)

A dice game in terminal

![](screenshot.png)

## Installation

Clone this repository and run:

```sh
cargo install --path .
```

## How to play

Just start the game:

```sh
yachtee
```
